Рабочий домен - python-django-blog.ru

Fitness diary - это дневник питания с информативными метриками и удобным интерфейсом добавления продуктов.

Главная страница - визуализация результатов месяца с кратким резюме каждого дня. 

Через календарь можно перейти в любой день месяца где можно добавлять продукты с указанием веса. 
Левая панель дня - список 25 наиболее часто добавляемых продуктов за последние 30 дней. 
Правая панель дня - список продуктов текущего дня.
